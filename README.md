# 🌌 Universal Semantic Engine (Template)

> **Ultimate Goal:** Beyond simple summarization, the AI swarm identifies extreme connective points within any unfamiliar data to automatically generate a perfect, semantic-based mind map topology.

This repository is a **pure template framework** for the LLM-Wiki ecosystem. 
It contains the engine code, AI rules, and Obsidian configurations, but **no research data**. It is completely empty and ready for your own academic papers!

## 📁 Structure
- `index.md`: The main dashboard tracking all sources and concepts.
- `log.md`: An automated log file recording the actions, linting passes, and autonomous updates made by the AI swarm.
- `raw/sources/`: Directory for original PDF papers. (Not tracked in Git for copyright/size reasons).
- `wiki/sources/`: Markdown summaries and bibliographic data automatically extracted from raw PDFs by the AI.
- `wiki/concepts/`: Core semantic hubs created by the AI to interlink multiple sources.
- `.agents/`: Contains the global system rules and `AGENTS.md` (the Constitution).
- `scripts/`: Contains Python scripts (Parsers, Academic Search APIs, Vector DB).
- `.obsidian/`: Pre-configured Obsidian vault settings featuring semantic color grouping (Sources = Blue, Concepts = Orange).

## 🚀 How to use
1. Clone this repository locally.
2. Open the folder as a Vault in **Obsidian**.
3. Summon your AI agent to begin ingesting PDFs into the `raw/sources/` folder. The AI will autonomously build your private semantic network!
